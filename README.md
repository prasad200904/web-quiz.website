# Quiz System – PHP + JS + SQL

An online quiz platform where users can take quizzes and view scores. Includes an admin dashboard for managing questions.

## ⚙️ Tech Used
- PHP
- MySQL
- JavaScript
- HTML/CSS

## 💡 Features
- User login system
- Admin panel to add/manage questions
- Timer-based quiz
- Score calculation
- Result page with answers

## 🛠 How to Run
1. Clone the repo
2. Import `quiz.sql` into your MySQL database
3. Set up database config in `config.php`
4. Run using XAMPP or WAMP
